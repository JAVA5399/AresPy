# AresPy

*AresPy* es un script de Python, diseñado para extraer audio de YouTube (*MP3* o *M4A*). Permite extraer el audio audio de videos individuales o listas de reproducción (playlists) completas de forma rápida.

## Características
- *Descarga versátil*: Soporta videos únicos y playlists.
- *Formatos*: Conversión a .mp3 o .m4a.
- *Interfaz gráfica*: Interfaz hecha con Tkinter ejecutada directamente en consola.
- *Orden*: Las canciones se guardan en la carpeta seleccionada.
- *Portapapeles*: Pega enlaces rápidamente desde el portapales.
- *Selector de carpetas*: Elige exactamente dónde guardar tu música en el explorador de archivos.

## Requisitos

Para que *AresPy* funcione, necesitas tener Python instalado y las librerías necesarias del proyecto.

- *Instalar librerías:* 
`pip install pytubefix moviepy pyperclip`
   


## Cómo usar
- Ejecuta el script:
`python AresPy_Downloader.py`

- Sigue las instrucciones en pantalla: pega el enlace de YouTube, selecciona el formato y elige la carpeta.
